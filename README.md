# 🎓 Student Management System

## 📖 Description

The Student Management System is a command-line based application developed using Python. It allows users to manage student records efficiently by performing basic operations such as adding, viewing, searching, updating, and deleting student data.

This project is designed to demonstrate fundamental programming concepts like file handling, data structures, and modular programming in Python.

---

## 🚀 Features

* ➕ Add new student records
* 📋 View all students
* 🔍 Search student by roll number
* ✏️ Update student details
* ❌ Delete student records
* 💾 Persistent data storage using JSON file

---

## 🛠️ Technologies Used

* Python
* JSON (for data storage)

## ▶️ How to Run the Project

1. Install Python on your system
2. Download or clone this repository
3. Open terminal in project folder
4. Run the following command:

   python main.py
